## SentinelDesk

SentinelDesk es una herramienta modular de análisis y diagnóstico técnico orientada a auditoría defensiva en entornos Windows.

Desarrollado por PaulNeja, fundador del grupo independiente de investigación en ciberseguridad PN Security.

---

## Descripción General

SentinelDesk ha sido diseñado como una herramienta de evaluación técnica local destinada a realizar análisis preliminares sobre configuraciones, servicios y comportamientos del sistema operativo.

Su objetivo es ofrecer una base estructurada para inspección técnica defensiva en entornos propios o debidamente autorizados.

El proyecto se encuentra actualmente en fase experimental avanzada. Aunque el comportamiento básico y necesario para su funcionamiento opera correctamente, la aplicación no está libre de errores menores, detalles visuales pendientes de optimización o comportamientos no críticos que serán corregidos progresivamente en futuras versiones.

---

## Estado del Proyecto

La versión actual debe considerarse funcional pero experimental.

Esto implica:

El núcleo lógico principal funciona correctamente.

Las funciones esenciales están operativas.

Pueden existir imperfecciones visuales.

Pueden presentarse errores no críticos.

Pueden producirse comportamientos inesperados en ciertos entornos.

El desarrollo continúa activo y se prevén mejoras estructurales, optimización interna y corrección de errores en versiones futuras.

--

## Modelo de Versionado

Se proyecta una separación clara entre dos ramas:

Versiones Estables (Stable)

Mayor nivel de validación.

Enfoque en confiabilidad.

Funcionalidades consolidadas.

Menor probabilidad de fallos.

Versiones de Desarrollo (DEV)

Carácter experimental.

Integración temprana de nuevas funcionalidades.

Mayor propensión a errores, comportamientos inestables o fallos inesperados.

No recomendadas para uso crítico o productivo.

Las versiones DEV podrán experimentar colapsos, inconsistencias o cambios estructurales sin previo aviso.

Alcance y Limitaciones

---

## SentinelDesk:

No reemplaza herramientas profesionales de auditoría certificadas.

No sustituye soluciones comerciales especializadas.

No constituye un framework completo de pruebas de penetración.

No debe considerarse una herramienta pericial forense.

Está concebido principalmente como una herramienta de análisis preliminar, exploratorio o educativo dentro de entornos controlados.

Su utilización debe realizarse bajo criterio técnico y responsabilidad individual.
